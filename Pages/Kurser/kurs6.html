USER EXPERIENCE DISIGN

Utbildningsnummer: 201514480-2 och 201514500-2

Fastställd av ledningsgruppen: 2017-08-08

Kurspoäng

40 YHP

Innehåll

Kursen ger de studerande avancerade kunskaper och färdigheter i att programmera och strukturera Javascript-projekt.

De studerande får kunskaper om att det finns olika Javascript-versioner och lär sig jämföra dessa samt får förståelse för deras olika kompabilitet med olika webbläsare och plattformar. De studerande ges goda kunskaper gällande att använda olika Javascript-bibliotek t.ex. JQuery.

De studerande ges goda kunskaper i hur ramverk fungerar samt kunna välja lämpliga tekniker utifrån projektspecifikationer.

Kursen innehåller programmering i Javascript, däribland objektorienterade implementationer i Javascript m.m. Kursen tar även upp hur man använder ramverk i Javascript samt hur man tar hjälp av verktyg som är relevanta för Javascript-fokuserad utveckling.

Kursens läranderesultat (mål)

Efter avslutad kurs ska den studerande ha:

Kunskaper om:

  1. Javascript, grundläggande syntax
  2. Javascriptsversioner och dess kompabilitet
  3. Hur Javascript körs i webbläsaren
  4. Javascript-bibliotek
  5. Javascript-ramverk
  6. DOM
  7. AJAX
  8. HTTP 1.1 / 2.0
  9. Websockets och polling

Färdigheter i att:

  1. Lösa programmeringsuppgifter med Javascript
  2. Använda Javascript för att skriva browser-tillämpningar
  3. Använda Javascript-bibliotek och Javascript-ramverk

Kompetenser för att:

  1. Tillämpa JavaScript som ett programmeringsspråk för att lösa olika typer av tillämpningar inom webbutveckling.
  2. Utveckla en webbapplikation baserad på ett Javascript-ramverk
